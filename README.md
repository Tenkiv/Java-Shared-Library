## Tekdaqc-JVM-Library

### Setup
Using Gradle add the Tenkiv Java Library to your projects dependancies and make sure Maven Central is added under your repositories.

```groovy
repositories {
    mavenCentral()
}

dependencies {
    compile group: 'com.tenkiv.tekdaqc', name: 'java-library', version: '2.1.2.3'
}
```
